# Fun with Xaviour

This project contains fun and interactive HTML, CSS, and JavaScript files.  
It's a simple web-based project for learning and experimenting with front-end coding.

## Files
- `index.html` - Main HTML file
- `style.css` - Styling for the project
- `script.js` - JavaScript for interactivity

## How to use
1. Clone this repository or download as ZIP.
2. Open `index.html` in your browser.
3. Enjoy the interactive fun!
